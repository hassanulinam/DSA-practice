# DSA Practice Plan - Week 6-8

## Advanced Topics

### Week 6 Goals:
1. Deep dive into advanced dynamic programming
2. Master graph algorithms
3. Understand advanced tree problems
4. Learn bit manipulation techniques

### Week 7 Goals:
1. Practice complex graph problems
2. Master advanced tree algorithms
3. Learn system design concepts
4. Practice optimization techniques

### Week 8 Goals:
1. Comprehensive review of all topics
2. Tackle challenging problems
3. Begin mock interviews
4. Prepare for behavioral questions

### Topics to Focus:

#### 1. Advanced Dynamic Programming (Week 6)
- 2D DP problems
- DP with bitmasks
- DP with recursion
- Longest increasing subsequence
- Edit distance

#### 2. Graph Algorithms (Week 6)
- BFS and DFS on graphs
- Shortest path algorithms (Dijkstra, Bellman-Ford)
- Topological sorting
- Union-Find
- Minimum spanning tree

#### 3. Advanced Tree Problems (Week 7)
- Binary search trees
- Tree serialization/deserialization
- Tree diameter
- Path sum problems
- Segment trees

#### 4. Bit Manipulation (Week 7)
- Basic bit operations
- Bit manipulation techniques
- XOR properties
- Subset generation using bitmasks

#### 5. Math and Geometry (Week 7)
- Mathematical concepts for algorithms
- Geometry problems
- Prime numbers and factorization
- Matrix operations
