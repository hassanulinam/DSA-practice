# DSA Practice Plan - Week 3-5

## Core Algorithm Techniques

### Week 3 Goals:
1. Master two pointer techniques
2. Understand sliding window approach
3. Learn fast and slow pointers for cycle detection
4. Practice merge intervals and top-k elements

### Week 4 Goals:
1. Deep dive into BFS/DFS traversal
2. Master basic dynamic programming concepts
3. Strengthen recursion and backtracking skills
4. Practice advanced backtracking problems

### Week 5 Goals:
1. Review and consolidate all core concepts
2. Tackle medium to hard problems
3. Begin practicing with mock interviews

### Topics to Focus:

#### 1. Two Pointers (Week 3)
- Basic two pointer technique
- Two pointer for sorted arrays
- Remove duplicates from sorted array
- Container with most water problem

#### 2. Sliding Window (Week 3)
- Fixed size sliding window
- Variable size sliding window
- Maximum sum subarray of size k
- Longest substring without repeating characters

#### 3. Fast & Slow Pointers (Week 3)
- Cycle detection in linked lists
- Finding middle of linked list
- Detecting cycle in linked list

#### 4. Merge Intervals (Week 4)
- Merge overlapping intervals
- Insert interval
- Interval deletion

#### 5. Top K Elements (Week 4)
- Find k largest/smallest elements
- Top K frequent elements
- Kth largest element in array

#### 6. BFS/DFS (Week 4)
- Tree traversal (inorder, preorder, postorder)
- Level order traversal
- Graph traversal

#### 7. Dynamic Programming (Week 4)
- Fibonacci sequence
- Climbing stairs
- House robber
- Maximum subarray

#### 8. Recursion and Backtracking (Week 5)
- Basic backtracking
- Permutations and combinations
- N-Queens problem
- Sudoku solver
