# DSA Practice Plan - Week 1-2

## Foundation Building

### Week 1 Goals:
1. Understand time and space complexity analysis
2. Master basic array and string operations
3. Learn linked list fundamentals
4. Practice stack and queue operations

### Topics to Focus:

#### 1. Arrays and Strings (Week 1)
- Array manipulation techniques
- String operations and manipulation
- Sliding window technique (basic)
- Two pointer approach (basic)

#### 2. Linked Lists (Week 1)
- Singly linked list fundamentals
- Linked list traversal
- Basic operations (insert, delete, search)
- Detecting cycles in linked lists

#### 3. Stacks and Queues (Week 1)
- Stack operations (push, pop, peek)
- Queue operations (enqueue, dequeue)
- Applications of stacks and queues

### Recommended Problems:

#### Arrays and Strings:
1. Two Sum (Easy)
2. Reverse String (Easy)
3. Valid Palindrome (Easy)
4. Move Zeroes (Easy)
5. Best Time to Buy and Sell Stock (Easy)

#### Linked Lists:
1. Linked List Cycle (Easy)
2. Remove Nth Node From End of List (Medium)
3. Reverse Linked List (Easy)
4. Merge Two Sorted Lists (Easy)
5. Palindrome Linked List (Easy)

#### Stacks and Queues:
1. Valid Parentheses (Easy)
2. Implement Stack using Queues (Easy)
3. Implement Queue using Stacks (Easy)
4. Min Stack (Medium)
5. Binary Tree Inorder Traversal (Easy)
